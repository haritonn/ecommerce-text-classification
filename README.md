== Что происходит

Классификация текстов на датасете ecommerceDataset (Kaggle). Используются:
* GRU;
* Основы NLP (Лемматизация, токенизация, удаление стоп-слов, добавление pos tags);
* collate_fn для корректной выдачи последовательностей DataLoader'ом;
* В остальном - "классический" пайплайн для обучения модели
